Smarter Parker Smart Parking Lot Monitor

Project Description

This Smart Parking Lot Monitoring (SPLM) system detects cars, pedestrians and bikers and catalogues them in the database. This can be significantly useful for law enforcement personnel as well as parking administration officials.


Project Details

  • Xilinx Ultra96 v2 Embedded FPGA Computer
  • YOLOv3 Convolutional Neural Network
  • Python3 with Xilinx Pynq DNNDK on-board
